# Icon component
The Icon component makes it easy to efficiently and accessibly incorporate icons into your user interface.

To use the Icon component, first ensure you have an **SVG sprite** file containing symbols for your icons. An icon is a visual representation of an object and can be used to represent actions or objects in the interface. Icons are created using **SVG**.

**Note:** \
Ensure that the **IDs** of the symbols in your **SVG sprite** file match the names of the icons you intend to use with the Icon component.

## How create a SVG sprite
```html:
<svg version="1.1" xmlns="http://www.w3.org/2000/svg" fill="currentColor" height="0" width="0">
  <symbol id="chevron-up" viewBox="0 0 24 24">
    <path d="..."/>
  </symbol>
  <symbol id="chevron-right" viewBox="0 0 24 24">
    <path d="..."/>
  </symbol>
<svg/>
```
## Usage
```
<Icon name="icon-name" size="m" title="Optional description" />
```
